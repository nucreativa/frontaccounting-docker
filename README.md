# FrontAccounting
This is an unofficial Git Repo for FrontAccounting.

## How to use
```
docker-compose up
```

add this line to your `/etc/hosts`
```
0.0.0.0 fa.dev
```
and open `http://fa.dev` in your browser and do the installation.

DB Settings
```
hostname: mariadb_master
username: dockertest
password: dockertest
dbname: dockertest
```
